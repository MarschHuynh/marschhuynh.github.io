Cách tạo DrawerLayout trong android

2016-04-20
  • android
  • layout

DrawerLayout

Sau đây là các bước tạo DrawerLayout

  1. Thêm các giá trị sau vào file res -> values -> string.xml

  2. Thêm các giá trị sau vào file res -> values -> color.xml

  3. Thêm các giá trị sau vào file res -> values -> dimens.xml nếu trùng thì bỏ qua nhé.

  4. Thêm style sau vào style.xml.

  5. Mở Manifest.xml và thay đổi theme thành MyMaterialTheme nhé

  6. Thêm toolbar (ActionBar)

    Tạo file res -> toolbar.xml(layout)

  7. Tạo menu item cho toolbar

    Tạo file menu_main.xml trong res -> menu

  8. Đổi file MainActivity.java có nội dung nhu sau.

  9. Tạo class NavDrawerItem.java

  10. Tạo layout nav_drawer_row

  11. Tạo một layout khác có tên fragment_navigation_drawer

  12. Chúng ta dùng RecyclerView để hiển thị các item trên drawer layout nhé, vì vậy tạo NavigationDrawerAdapter.java có nội dung như sau

  13. Tạo FragmentDrawer.java đây chính là DraweLayout

  14. Bây giờ thay đổi layout của MainActivity

  15. Thay đổi MainActivity.java để thêm DraweLayout

Bây giờ run là bạn đã có thể thấy kết quả rồi nhé, bước tiếp theo mình sẽ hướng dẫn các bạn thêm fragment và chuyển qua fragment khác khi kích vào DrawerLayout item

  1. Bây giờ chúng ta tạo một fragment có tên là HomeFragment.java

  2. Thay đổi MainActivity.java có nội dung như sau
